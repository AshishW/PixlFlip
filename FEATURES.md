PixlFlip is frontend of full stack application which allows users to create AI powered Flash cards for revision. It's powered by Gemini's Advanced AI models. 

## Features
- Generates flash card decks based on topic or user input. 
- credits based system that shows how many credits are left for user to generate flash card decks.
- user can manually edit the decks as well. 
- visual progress bars and session tracking when studing a card deck. 
- full hotkey support for studying session. 
- Gamified experience with points, badges for number of flash cards generated and studied.

## UI
- Dashboard with all the decks and their progress. top right navigation icoons for profile (login/logout), credits, help. modern centered Input field for topic name and generate button.
- Gamified UI look that encourages learning experience. 
- Card decks are displayed as card deck design with title, description, progress and buttons to edit, study, delete.
- library management system to manage the decks in dashboard itself. 
- user can interact with cards during study session, like flipping (animation), editing, deleting.
- user can study the cards in a session, where they can study multiple cards at once.\
- victory screen after all cards are studied with some animation and points. 
- user can logout and login.
